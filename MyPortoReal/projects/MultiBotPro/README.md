## Features

- **Economy System** - Balance, daily reward, work, leaderboard
- **Leveling System** - XP per message, level up notifications
- **Moderation** - Kick, ban, mute, warn
- **Music Player** - Play, queue, skip, loop

## Installation

```bash
git clone https://github.com/fajrifajar/multibot-pro.git
cd multibot-pro
npm install
cp .env.example .env
npm start
```

## Tech Stack

- Node.js
- Discord.js v14
- MongoDB
