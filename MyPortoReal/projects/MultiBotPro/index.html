Fitur

4 Sistem Dalam 1 Bot

Economy

Balance, daily reward, work system, dan shop

Leveling

XP per message, level up, dan leaderboard

Moderation

Kick, ban, mute, warn dengan logging

Music

Play, queue, skip, loop dengan YouTube

Commands

Daftar Perintah

Economy

!balance - Cek saldo
!daily - Reward harian
!work - Kerja dapat uang

Leveling

!level - Cek level
!leaderboard - Top users

Moderation

!kick - Kick member
!ban - Ban member
!mute - Mute member

Music

!play - Play musik
!skip - Skip lagu
!stop - Stop musik
Instalasi

Cara Install

# Clone & Install

$ git clone https://github.com/FaaWeell/multibot-pro.git

$ cd multibot-pro

$ npm install

# Setup & Run

$ cp .env.example .env

$ npm start
