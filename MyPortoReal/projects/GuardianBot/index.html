Guardian Bot

Bot keamanan Discord untuk melindungi server dari spam, raid, dan aktivitas berbahaya. Dengan sistem auto-moderation yang cerdas.

Proteksi

Sistem Keamanan

Anti-Spam

Deteksi dan hentikan spam secara otomatis dengan sistem warning bertingkat.

  • Deteksi pesan berulang
  • Auto-mute spammer
  • Bulk delete spam

Anti-Raid

Proteksi dari serangan raid dengan deteksi mass-join dan auto-lockdown.

  • Deteksi mass join
  • Auto kick raiders
  • Lockdown otomatis

Auto-Mod

Moderasi otomatis untuk kata terlarang, mention berlebihan, dan link berbahaya.

  • Filter kata terlarang
  • Limit mention
  • Link whitelist
Commands

Daftar Perintah

Moderation

!warn Beri warning ke member
!warn @user [alasan]
!kick Kick member dari server
!kick @user [alasan]
!ban Ban member dari server
!ban @user [alasan]
!mute Mute member untuk durasi tertentu
!mute @user [durasi]
!purge Hapus banyak pesan sekaligus
!purge [jumlah]
!lockdown Lock atau unlock channel
!lockdown [off]

Utility

!help Tampilkan daftar command

Instalasi

Cara Install

terminal

# Clone repository

$ git clone https://github.com/FaaWeell/guardian-bot.git

# Masuk ke folder

$ cd guardian-bot

# Install dependencies

$ npm install

# Setup environment

$ cp .env.example .env

# Jalankan bot

$ npm start

Konfigurasi

Edit config.js untuk mengatur:

antiSpam Max messages, time window, mute time
antiRaid Max joins, lockdown duration
autoMod Banned words, max mentions, link whitelist
