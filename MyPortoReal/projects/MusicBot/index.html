MusicMaster Bot

Bot musik Discord dengan kualitas audio tinggi. Support YouTube, Spotify, dan SoundCloud dengan fitur queue management yang lengkap.

Install Bot Lihat Commands
Fitur

Yang Bisa Dilakukan

YouTube

Play musik langsung dari YouTube dengan kualitas audio terbaik

Spotify

Support Spotify tracks dan playlists dengan auto-search

SoundCloud

Akses jutaan track dari SoundCloud creators

Queue System

Kelola antrian musik dengan shuffle, loop, dan skip

Audio Filters

Bassboost, nightcore, vaporwave, dan filter lainnya

Volume Control

Atur volume dari 0-200% sesuai kebutuhan

Commands

Daftar Perintah

!play Play musik dari URL atau keyword
!play [url/query]
!pause Pause musik yang sedang diputar
!resume Lanjutkan musik yang di-pause
!skip Skip ke lagu selanjutnya
!queue Lihat antrian musik
!volume Atur volume musik
!volume [0-200]
!loop Loop track/queue
!loop [track/queue/off]
!shuffle Acak urutan queue
!nowplaying Info lagu yang sedang diputar
!stop Stop dan clear queue
Instalasi

Cara Install

terminal

# Clone repository

$ git clone https://github.com/FaaWeell/musicmaster-bot.git

# Masuk ke folder

$ cd musicmaster-bot

# Install dependencies

$ npm install

# Setup environment

$ cp .env.example .env

# Jalankan bot

$ npm start

Environment Variables

BOT_TOKEN Token dari Discord Developer Portal
CLIENT_ID Client ID bot kamu
LAVALINK_HOST Host Lavalink server
